AI 에이전트

스스로 다음 행동을 정해 실행한다
  • 목표를 주면 스스로 계획·판단함
  • 필요한 도구를 직접 호출 (검색·API)
  • 여러 단계를 스스로 실행·반복함
스스로 판단해 실행한다
학술적 정의(Russell & Norvig): "환경을 인식하고 목표를 위해 행동을 스스로 선택하는 시스템" — 오늘날 AI 에이전트는 여기에 LLM 두뇌 + 도구가 결합된 형태
핵심 = 목표를 위해 스스로 다음 행동을 결정(자율) + 도구 사용  ·  정해진 순서만 따르는 '자동화'와 달리 경로를 스스로 정함
🧠 판단(추론)🔌 도구 사용🔁 다단계 실행💾 기억·⏰선제 (선택)
챗봇이 '대답'이라면, 에이전트는 '실행'이다

에이전틱 워크플로우 (Agentic Workflow)

에이전트가 목표를 이루는 동작 원리 — 입력 → 의사결정 → 실행 → 피드백
① 입력
Input
사용자가 질문이나 데이터를 전달한다
예) 식단 사진 전송
예) 학부모 민원 접수
▶
② 의사결정
Decision
질문을 분석해 어떤 도구를 쓸지 스스로 판단
예) 이미지 분석이 필요?
예) 웹 검색이 필요?
▶
③ 실행
Execution
외부 도구·지식에 접근해 데이터를 수집·처리
예) Gemini에 분석 요청
예) Tavily 웹 검색
▶
④ 피드백
Feedback
결과를 검증하고 최종 답변을 생성한다
예) 칼로리 분석 결과 회신
예) 종합 민원 답변 작성
🔁 필요하면 이 네 단계를 다시 반복 (Loop)
입력 → 의사결정 → 실행 → 피드백을 반복하는 것이 에이전틱 워크플로우
실습(1) 다이어트 코치한 걸음 더 · 대한중학교 봇
두 실습 모두 이 흐름을 따른다
